About the OSS

The Observatory Status System, or OSS, is designed to provide a database of astronomical observing facilities across the world, with live updates regarding the status and scheduling of instruments and facilities wherever possible. It was developed by Rachel Street and the TOM toolkit team, building on the idea of a shared spreadsheet of facilities hosted by the Treasuremap system (credit: Samual Wyatt, Aaron Tohuvavohu, Jamison Burke and the Treasure Map development team).

Please note the system is still in development!

The data provided by this system are intended to be publicly available to help observers to coordinate observing programs. The system offers both an online user interface as well as an Application Programmable Interface.

A package of commandline tools to access the API are provided by the accompanying Observatory Status Tools package.

Wherever possible the status of different observing facilities are updated automatically once a day by programmatic query. As this is not possible for all facilities, telescope operators are invited to apply for an account, which would enable them to update the status of their telescopes and/or instruments, either through this UI, or by using the commandline tool in the Observatory Status Tools package. To apply for an account, please contact: rstreet (at) lco.global.
